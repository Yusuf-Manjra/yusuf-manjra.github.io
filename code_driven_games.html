Basic Unreal Engine AI FPS Arena

‘A basic first person target shooter with an AI opponent patrolling around and chasing you if it sees you.’

The player has a basic gun that has 2 firing inputs, left click fires a fast moving projectile that deals 20 damage, right click fires a slow moving projectile that deals 50 damage.
The game contains a basic map with a navmesh placed over it to allow for the AI to function. The map also contains 2 shootable objects, a physics object that bounces around the room when shot at based on the velocity of the impact and a floating orb that has 200 health and will respawn at a random position in a given radius around an initial spawn point.
Hitting the physics object with the fast projectile gives 2 points whilst hitting it with the slow projectile gives 1 point. Destroying the orb gives 5 points.
If the AI enemy comes into contact with the player the game will reset. The enemy AI cannot be destroyed but can be pushed around by the players slow projectile.


Obstacle Avoidance Game Using Camera Controls

‘An obstacle avoidance game where you use coloured cards to move the player around to survive as long as possible while avoiding the walls coming towards you.’

Grey walls are impervious and give one point for passing. The coloured walls allow you to pass through them if you are the same colour as them and will give multiple points for passing through them.
Falling off screen will kill the player and end the game.

The game uses a dynamic-linked library that contains the functions needed to read the camera using the OpenCV library and then perform calcuations that are needed within the game such as colour and location detection.
